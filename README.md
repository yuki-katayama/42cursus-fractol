# Fract-ol

## Summary
This project is meant to create graphically beautiful fractals.

### mandatory part

valid param : mandelbrot / julia
+ ESC : Exit program.
+ Mouse scroll : Zoomin and zoomout.

### bonus part
valid param : mandelbrot / julia 
 / burningship

+ ESC : Exit program.
+ K : Zoom in
+ L : Zoom out
+ Mouse scroll : Zoomin and ​zoomout.
+ E / D : Contorol divergence value
+ W / S : Contorol depth 
+ R / G / B : Change color
+ Z / X : Contorol actual number
+ C / V : Contorol image number
+ ← / → / ↑ / ↓ : Moving cursol
+ I : Reset
